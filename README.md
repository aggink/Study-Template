# Реализация задания 1 (Вариант 7) — Класс MySet

Этот PR содержит реализацию Лабораторной работы 1, Задание 1, Вариант 7.  
В рамках задания создан обобщенный класс `MySet<T>`, представляющий математическое множество, содержащее только уникальные элементы.

## Что реализовано

- Создан класс `MySet<T>` для работы с множествами.
- Для хранения элементов используется `HashSet<T>`, что гарантирует уникальность значений.
- Все операции над множествами возвращают новые объекты и не изменяют исходные множества.
- Реализована перегрузка операторов:
  - `|` — объединение множеств;
  - `&` — пересечение множеств;
  - `-` — разность множеств;
  - `/` — симметрическая разность;
  - `==` и `!=` — сравнение множеств по значению.
- Переопределен метод `ToString()` для вывода множества в формате `{elem1, elem2}`.
- Добавлена демонстрация работы класса в методе `RunService.RunTask1()`.

## Unit-тесты

Для проверки корректности реализации написаны unit-тесты в файле `MySetTests.cs`.

Проверяются следующие сценарии:

- создание множества с уникальными элементами;
- удаление дубликатов при инициализации;
- корректная работа объединения;
- корректная работа пересечения;
- корректная работа разности;
- корректная работа симметрической разности;
- сравнение множеств по значению;
- корректный вывод множества через `ToString()`.

## Тестирование

Все unit-тесты проходят успешно.

## Влияние на проект

Изменения локализованы в рамках `Task1` и не затрагивают другие модули проекта.
